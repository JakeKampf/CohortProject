About

  • This project was designed to create schedules for various cohorts at WMU
  • To Generate Schedules, First enter in all class requirements on the requirements page.
    • Cohort represents the name of the cohort, and should match all other requirements for that cohort.
    • Class is the name of the needed class, it should be entered as all uppercase letters and no spaces. (ie ECE1100)
    • Seats required is the number of seats needed in that class by the cohort.
    • Prefered Sections is the section prefix that is prefered for the cohort. (for sections like H01, enter H, for no preference leave blank)
    • Ensure all necessary requirements are entered before running the scheduler.
  • The Schedules page allows you to run a schedule and view past generated schedules.
    • The Number to Generate box indicates how many potential schedules should be generated, this should be kept below 20 to reduce clutter and duplicates
    • The Schedule Name should indicate the semester that you are running the scheduler for, and should not match a the name from a prevous run
    • The file upload should be an excel file (.xlsx). It should contain a row for each class section which can be chosen
